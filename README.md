# mechama.eu — Landing Page Institucional

Site institucional do domínio principal `mechama.eu`, criado para fortalecer a presença da marca MeChama e aumentar as chances de aprovação no Google AdSense.

## Stack

- **React 18** + Vite
- **Tailwind CSS 3**
- **Lucide React** (ícones)
- i18n manual com suporte a **PT-BR, EN, ES e FR**

## Páginas e seções

- Hero — apresentação da marca
- Sobre Nós — quem é a MeChama
- Produtos — Rodízio Race + Coming Soon
- FAQ — perguntas frequentes
- Contato — formulário de contato
- Política de Privacidade
- Termos de Uso

## Como rodar

```bash
npm install
npm run dev
```

## Deploy (Vercel)

```bash
npm run build
# deploy da pasta dist/
```

## Variáveis de ambiente

Nenhuma necessária por enquanto.
